Block Party

Block Party is an upcoming physics-based block-stacking 2D puzzle-platformer for Nintendo Wii U. The objective of the game is to ascend to the top of the level, working your way up by building a tower of blocks with the Wii U Gamepad. Building the tower is only half the puzzle, as the only way to get to the top is to climb the tower as you make it! Every block has some special property you can use to get to the top, every level has a limited set of available blocks, and there are a variety of solutions for each level. Block Party is being developed in Javascript for the Nintendo Web Framework and is expected to be launched in 2015.

  • Project Type: Personal (Group of 2)
  • Date: May, 2014 to Present
  • Platforms: Nintendo Wii U (Unreleased)
